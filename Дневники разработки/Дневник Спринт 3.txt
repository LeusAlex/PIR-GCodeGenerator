Спринт 3. 
Выбран тип станков для обеспечения совместимости с приложением - 3-х осевые ЧПУ фрезеры на логике GRBL (Ардуино-подобные микроконтроллеры). 
Изучены основные команды G-кода под выбранный станок. Исключены из работы команды круговой подачи (G2 G3). 
Сформирован перечень необходимых траекторий:
Обработка по прямоугольной спирали для плоскости, встречное фрезерование. 
Пошаговая обработка одного уровня для отверстия, возможна реализация не вертикального заглубления фрезы для снижения вероятности аварий. 
Для резьбы планируется реализовать все основные параметры: правая/левая резьба, направление фрезерования (попутное или встречное) с учетом постоянного вращения фрезы по часовой стрелке. 
Начинаю прописывать логику расчета координат точек на траектории.
